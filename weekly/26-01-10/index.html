完成什么事儿，让我觉得 2026 值了
#

Chris Williamson 在新访谈中提到了一个很重要的事实：回溯过去，重要的事情很少。既然如此，我们在计划生活的时候，应该一开始就关注在这些 事情上。关注真正应该做的事情上。不要计划在新年多达成 10 个目标，我们不能比现在做得更多，能做的其实是筛检，问自己今年应该关注在哪两三件事上。

This is the opportunity to become anything you want behaviorally, but you can’t become everything you want. So you need to pick a small number. Ask yourself the question: What would have to happen by the end of the year for me to look back on the year and consider it a success?

It usually comes down to a few things.

Setting the bar unrealistically high is not increase your performance. In order to pick something up, you need to put something down. Make the assumption that I can do no more than I am doing now. I can switch staff, but I can not add more in.

我想我的答案是：

  1. 加薪
  2. 记录
  3. 关注自我

对我来说今年最重要的事情就是加薪。这是一箭三雕的事。首先额外的收入允许我更多地达成与生活的连接，去更多的体验和接触；然后薪酬体现了社会对我工作的认可度， 以及我对自己的能力的信心；加薪也让积蓄更多，更早的 FIRE，让自己有更多的选择权。

当然，加薪是结果不是达成的手段。去年一是做了太多别人想要，但对我不重要的事情；二是目标太多，太过贪心，反而什么也没达成。今年我选择全力投入到工作能力上去， 把产品功能的各个主要模块都摸清楚。这样不论现公司要涨薪还是跳槽，我都能够做到。而且之后的工作中也能够去承担最核心的任务，避免像以前一样做一些边角料的活。

记录也是今年必须要完成的事情。记录的内容很宽泛：我的想法，我的工作，我的阅读，我的生活，一切的一切我都想记录。这看起来很贪心，不过我不要求自己事事不漏， 而是通过记录一切来养成记录的习惯。过去的三年间，我的记录太少，不论是工作还是生活上，这都给我带来了很多的困扰。没有工作记录，事后就没有复盘和改善，导致 重复时间，加深错误。生活上没有记录，回过头来，不知道时间哪里去了，像是一个游魂。

我们总是觉得人生还未开始，“等我实现xxx之后，一切就好了”。但这是假的，达成了一个目标还有下一个目标，我们只是不断地从一个边缘到另一个边缘，永远无法达到中心。 记录是帮助我活在当下的工具。现在就是我们的生活，用记录帮我们修正航向。

最后是关注自我。去年的不如意，很大程度是因为自己关注太多周围的环境，关注别人的工作，而让自己陷入了无谓的投入中去。害怕遗漏每一个消息，担心会有任何的不利； 承担自己不愿意做，对成长没帮助的工作，因为别人这个给安排了工作，或者没有其他人愿意做。 这些担心都是虚假的，或者说，自己对外界的敏感反而让真正该担心的事情成真。今年，我只关心自己。当然这不是自私，而是更加专注和积极。去积极争取自己想要的东西， 然后专注在这些真正重要的事情上。而不是捕风捉影，浪费精力。

学了点儿

vim 相关
#

  • vim 自带命令

    • vim r 命令读取 cli output 很有用。比如 fio 测试 dump 了一堆 mismatch file，我们想要知道具体有哪些位置，就可以用 r! ls 把文件名都导入，然后去除前后缀， 就得到了 offset 位置。还可以继续利用其他的 cli 命令，比如去重等等。
    • vim 替换命令中 .* 为搜索对象为所有对象，& 为被匹配的对象。vim 是基于 ed 的，所以操作对象是 row。基于这个特点，我们可以容易的为目标添加前后缀。 比如，我想给文件夹内的所有 markdown 文件添加一个目录，就可以先用上面的技巧，把所有文件名都读入，然后用 :%s/.*/[&](&)/ 添加 link。

  • neovim 图片支持

    • snacks.nvim 的 image 功能支持图片预览，还有 tex 数学公式的渲染
    • img-clip.nvim 方便插入剪贴板中的图片，也支持拖拽插入。还可以利用自定义命令对图片进行操作，比如压缩、旋转等等之后再插入

Linux 相关
#

  • Linux
    • vial 在 linux 上需要添加 udev 规则来配置权限，但是文档里给的规则不一定好用。原因是有些 distro 上面需要另外配置 user 的值，比如 fedora 需要改为用户名， 而 arch 改成用户名也不行。但是根据规则内容其实是配置读写权限，我想可能是用户权限不足，尝试将 user 改为 wheel 管理组后，可以正常使用了。

粗野派
#

阿德里安布罗迪是我很喜欢的演员。之前电影宣发的时候就挺感兴趣，也看了一点儿开头，因为太长，没看完，这周末找时间看了。 片子值得一看，采石场和建筑的布景都很美，所以我也选了采石场的这张宣发图。

整个故事的走向并不传统，之前看开头，以为是那种换个地方重开，被伯乐相中，然后开始壮大的路子。但实际上故事很平淡，甚至可以说是没有 特别的起伏，但是又一切在情理之中。随手搜了一下，原来影片主旨讲得就是美国移民的困境，是那些被迫远离他乡，在外却又不被接纳的 美国移民缩影。
