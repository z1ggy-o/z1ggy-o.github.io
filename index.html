BCW: Buffer-Controlled Writes to HDDs for SSD-HDD Hybrid Storage Server

Guangyu Zhu published on 2020-12-31 included in papers
Short Summary HDDs are under utilized in hybrid cloud storage systems which makes SSD to handle most of the requests. This shorts the life of SSDs and also wasts the utilization of HDDs. The authors of this paper find that the write requets can have $μ$s-level latency when using HDD if the buffer in HDD is not full. They leverage this finding to let HDD to handle write requests if the requests can fit into the in disk buffer.
